Тег Div – это блоковый элемент, который обычно занимает всю ширину экрана.

Тег Span – это строчный элемент, который занимает только столько места, сколько нужно для его содержимого.

Статья о HTML элементах

В HTML есть два типа элементов — блоковые и строчные

Тег Div - пример блокового элемента, который используется для создания крупных элементов на странице.

Тег Span - пример строчного элемента, который используется для выделения текста внутри абзаца.

Заключение

Понимание разницы между блоковыми и строчными элементами помогает создавать четко структурированные веб-страницы.
